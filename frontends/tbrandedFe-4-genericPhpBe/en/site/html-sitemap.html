Sitemap of the OSCAd - t-branded interface

  • OSCAd :- About OSLiC, the Open Source Compliance Advisor, sponsored by Deutsche Telekom AG
    • Teaser :- The manifesto of OSCAd, the Open Source Compliance Advisor, sponsored by Deutsche Telekom AG
    • License :- The licensing of OSCAd, the Open Source Compliance Advisor, sponsored by Deutsche Telekom AG
    • Contributions :- The initiators of OSCAd, the Open Source Compliance Advisor, sponsored by Deutsche Telekom AG
    • Manifesto :- The manifesto of OSCAd, the Open Source Compliance Advisor, sponsored by Deutsche Telekom AG
  • REQUESTs
    • via OSLiC form sheet :- Get answers of the Open Source Compliance Advisor automatically computed in the base of a filled form sheet
    • via OSUC taxonomy :- Get answers of the Open Source Compliance Advisor automatically computed on the base of traversing the oslic finder
    • via Simplification :- Ask the Open Source Compliance Advisor to get way to simplify the license fulfilling compliance work
    • Understanding the parameters
      • 'Type' :- explains which kind of software is referred by which marker
      • 'Context' :- explains in which contexts a piece of open source software can be used
      • 'State' :- explains in which states open source software can be used
      • 'Recipient' :- explains in which way the recipients of open source software are clustered by the OSLiC / OSCAd
      • 'Form' :- explains in which forms open source software can be distributed
  • DOWNLOAD
    • prepare :- How to prepare the download and the use of the source for developing the Open Source Compliance Advisor
    • obtain :- How to obtain / fetch the OSCAd sources for developing the Open Source Compliance Advisor
    • install :- How to install the development environment of Open Source Compliance Advisor
