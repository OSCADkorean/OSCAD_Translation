Context Rules

  • A program (proapse) is ever used / distributed as an independent unit [but not vice versa]
  • Each embedded component is a snippet, module or library (snimoli) [but not vice versa].
  • Each snippet, module or library (snimoli) being used 4yourself is an embedded component [but not vice versa].
